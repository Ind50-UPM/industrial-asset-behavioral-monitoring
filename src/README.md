# Source Packages

This directory contains the implementation packages and generated artifacts used by the repository.

## Main Areas

- `Model_A/`: reusable package for elementary operating-state identification, published locally as `iabm`
- `models/`: persisted model artifacts generated by training workflows
- `predictions/`: validation reports and prediction outputs generated by experiments

Future package-level implementations for `Model_B` and `Model_C` can follow the same structure as `Model_A`.
