# Model_B: Behavioral Sequence Analysis

`Model_B` is the second modeling layer in the Industrial Asset Behavioral Monitoring framework. It transforms state timelines into contiguous state runs, active behavioral sequences, and sequence-word summaries that can later support nominal-pattern modeling and anomaly analysis.

## Scope

This package currently provides:

- loading of state timelines from Excel, CSV, or Parquet
- support for `Model_A` prediction outputs and digital state traces
- smoothing of short transient runs
- extraction of contiguous state runs
- extraction of higher-level active sequences
- summary tables for repeated sequence words
- comparison of observed sequences against nominal sequence references
- DTW-style alignment distance and explicit anomaly flags
- bilingual CLI messaging through English and Spanish catalogs

## Installation

```bash
cd industrial-asset-behavioral-monitoring/src/Model_B
poetry install
```

## Command-Line Usage

Analyze a prediction file generated by `Model_A`:

```bash
poetry run industrial-behavior \
  --input ../predictions/Model_A/predictions_analogicas_nonans.parquet_RF.xlsx \
  --output-dir ./reports \
  --smooth-short-runs
```

Compare an observed timeline against a nominal reference timeline:

```bash
poetry run industrial-behavior \
  --input ./observed_predictions.xlsx \
  --nominal-input ./nominal_predictions.xlsx \
  --output-dir ./reports \
  --anomaly-threshold 1.5
```

## Outputs

The CLI writes three reports:

- `state_runs.xlsx`: contiguous runs of the same state
- `active_sequences.xlsx`: grouped non-zero state sequences
- `sequence_words.xlsx`: repeated sequence patterns and average durations
- `sequence_comparison.xlsx`: anomaly-oriented comparison against nominal words when `--nominal-input` is provided

## Python API

```python
from iabm_behavior import BehavioralSequenceAnalyzer

analyzer = BehavioralSequenceAnalyzer(state_column="Predicted_State")
timeline = analyzer.load_state_timeline("predictions.xlsx")
runs = analyzer.extract_runs(timeline)
sequences = analyzer.extract_active_sequences(timeline)
summary = analyzer.summarize_sequence_words(sequences)
nominal_reference = analyzer.build_nominal_reference(sequences)
comparison = analyzer.compare_to_nominal(sequences, nominal_reference)
```
